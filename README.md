# Morse_Code
<h2>Text to morse code converter</h2><br>
<div style="text-align:center">
This is a simple text to morse code converter.<br>
Users may input a message (English Aplhabet and Punctuation is accepted).<br>
The program will then return a morse code representation of the text.<br>
The user then has the option of playing the morse code message.
</div>
